# Cat-Jokers
Cat Jokers for Balatro. Adds -- Jokers, 1 Deck, 1 Challenge, and 2 Blinds. Cat Jokers are most powerful when used in big groups, so most add Chips or Mult based off of how many other Cat Jokers there are, while some make it easier to get other Cat Jokers.

# Jokers:
| Name        | Effect              | 
| ----------- |:-------------------:| 
| Siamese     | +40 Chips per Cat   | 
| Persian     | +4 Mult per Cat     | 
| Shorthair   | x1.5 Mult per Cat   |
| Siberian    | Retriggers Cats     |
| Sphynx      | Adds random Cat     |
| Mother      | Adds Negative Kitty |
| Kitty       | Does nothing        |
